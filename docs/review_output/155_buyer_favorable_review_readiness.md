# 155. Buyer-favorable Review Readiness

## 평가 기준
- 1) 당사 보호 방향으로 검토하는가: 부분 도달(heuristic posture + 일부 룰 패치)
- 2) 계약유형별로 질문이 달라지는가: 부분 도달(계약 텍스트/조항 기반 상위 3~5개)
- 3) 조항 매핑이 정확한가: 부분 도달(불일치 시 docx 생성 실패로 방어)
- 4) 적용 법령이 계약유형과 맞는가: 부분 도달(컨텍스트 기반 필터/리랭크, 대리점법 우선 토픽 조건부 제거)
- 5) 수정문안이 clause-specific하고 실무적으로 타당한가: 부분 도달(템플릿 복붙 제거 + 패치 기반)
- 6) Word redline이 실제 법무 검토용으로 쓸 만한가: 부분 도달(변경 조항만 출력 + 변경 부분만 색/취소선)
