# 262. Strategic Legal Counsel 모드 작업 항목 (퍼시스 수급인 전략)

## 1. 입력/전처리
- 계약서 전체를 읽고 퍼시스의 지위를 **수급인(을)** 으로 확정한다.
- 목적(제1조), 정의(제3조) 조항에 실무 의무(비용, 안전, 하도급 등)를 삽입하는 자동 수정안은 생성하지 않는다.

## 2. 핵심 선별 로직(Top 5 Picks)
- 계약서 전체에서 “퍼시스에게 실질적으로 돈이 나가는” 독소 조항 5개만 선별한다.
- 최소 포함해야 하는 카테고리:
  - 지체상금(일 0.3% → 0.1% 이하)
  - 상계/공제(확정 채권 + 사전 서면합의)
  - 해지(30일 이상 서면 최고/시정)
  - 안전 책임(발주자 제공자료/현장 하자 면책·감경)

## 3. 수정문안 생성 규칙
- 각 Pick에 대해 아래를 포함한 “최종 수정 문안”을 작성한다.
  - 지체상금: 일 0.1% 이하 + 발주자 귀책/불가항력 예외 + 공기 연장/추가비용 정산 연결
  - 상계/공제: 확정 채권 요건 + 사전 서면합의(사유·금액·산식) + 정산서/증빙 제공 + 이의제기 기간
  - 해지: 위반사항 특정 + 30일 이상 서면 최고 + 시정기회 + 예외적 즉시해지 사유를 좁게 열거
  - 안전: 수급인 통제 가능한 영역으로 책임 한정 + 발주자 제공 자료/현장 기존 하자 귀책 명시 + 상호 협력 구조

## 4. 가드레일
- 조항 제목/주제와 무관한 쟁점의 끼워넣기를 금지한다.
- 동일 가이드를 여러 조항에 반복하지 않는다(핵심 조항 1곳에서만 강하게 제시).

## 5. Self-Audit(출력 전 최종 점검)
- 지위(수급인) 관점에서 불리한 조항을 그대로 승인한 항목이 있으면 전면 수정한다.
- 목적/정의 조항 오염(비용/안전/하도급 의무 삽입)이 남아 있으면 삭제한다.
- 지체상금 0.3%를 발견했는데도 0.1% 이하 조정안을 제시하지 않았다면 부적합으로 처리한다.
