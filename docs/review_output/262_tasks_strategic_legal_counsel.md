# 262. Strategic Legal Counsel 모드 작업 항목 (퍼시스 그룹 포지션 기반)

## 1. 입력/전처리
- 계약서 전체를 읽고 퍼시스 그룹의 포지션을 아래 중 하나로 확정한다.
  - 판매대리점/유통: 공급업자(갑)
  - 퍼플식스 인테리어: 수급인(을)
  - 가구 렌탈: 렌탈업자
- 목적(제1조), 정의(제3조) 조항에는 비용/안전/정산 등 실무 의무를 삽입하는 수정안을 생성하지 않는다.
- 동일 리스크 가이드를 여러 조항에 반복하지 않는다(핵심 조항 1곳에서만 제시).

## 2. 핵심 선별 로직(Top 5 Picks)
- 계약서 전체에서 “돈과 권리”에 직결되는 독소 조항 5개만 선별한다(HIGH 5개 이내).
- 계약 유형별 최소 포함 카테고리:
  - 수급인(인테리어/공사): 지체상금(0.3% → 0.1% 이하), 상계/공제(확정 채권+사전합의), 해지(30일 서면 최고), 안전(발주자 제공자료/현장 하자 면책·감경)
  - 유통/대리점: 가격 결정권 침해(승인→협의), 경영간섭, 정산/공제 남용
  - 렌탈: 소유권/임의처분 금지, 중도해지 위약금 상한, 청약철회/해지 절차 방해 금지, 추심/신용정보 동의·고지

## 3. 수정문안 생성 규칙
- 각 Pick은 해당 조항의 법적 기능/분쟁 포인트에 맞춰 최소 변경으로 조정한다(무분별한 덧붙임 금지).
- 수급인(인테리어/공사):
  - 지체상금: 일 0.1% 이하 + 발주자 귀책/불가항력 예외 + 공기 연장/추가비용 정산 연결
  - 상계/공제: 확정 채권 요건 + 사전 서면합의(사유·금액·산식) + 정산서/증빙 제공 + 이의제기 기간
  - 해지: 위반사항 특정 + 30일 이상 서면 최고 + 시정기회 + 예외적 즉시해지 사유를 좁게 열거
  - 안전: 수급인 통제 가능한 영역으로 책임 한정 + 발주자 제공 자료/현장 기존 하자 귀책 명시 + 상호 협력 구조
- 유통/대리점:
  - 가격: 판매가격 승인 강제를 “협의/권장” 구조로 변경하고 최종 결정권이 대리점에 있음을 명확화
- 렌탈:
  - 소유권: 렌탈 기간 소유권은 퍼시스에 존속 + 임의 처분/양도/담보 제공 금지
  - 위약금: 중도해지 위약금 과도 방지(상한) + 산식/근거 투명화
  - 추심/신용정보: 고지·동의·위탁관리 전제

## 4. 가드레일
- 조항 제목/주제와 무관한 쟁점의 끼워넣기를 금지한다.
- 동일 가이드를 여러 조항에 반복하지 않는다(핵심 조항 1곳에서만 강하게 제시).

## 5. Self-Audit(출력 전 최종 점검)
- 포지션과 반대 방향의 독소조항을 그대로 둔 항목이 있으면 전면 수정한다.
- 목적/정의 조항 오염(비용/안전/하도급 의무 삽입)이 남아 있으면 삭제한다.
- 지체상금 0.3%를 발견했는데도 0.1% 이하 조정안을 제시하지 않았다면 부적합으로 처리한다.
- HIGH 등급이 5개를 초과하면, 우선순위를 재정렬해 5개로 축소한다.
